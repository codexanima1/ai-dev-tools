# AI Developer Tools

Colección de scripts y herramientas para desarrolladores.

## Scripts

### ai_prompt_generator.py
Generador de prompts optimizados para diferentes tareas de programación.

```python
python ai_prompt_generator.py
```

Genera prompts para:
- Explicación de código
- Refactorización
- Debugging
- Documentación

### github_stats.py
Estadísticas y métricas de GitHub.

```python
python github_stats.py <username>
```

Muestra:
- Contribuciones recientes
- Lenguajes más usados
- Repositorios destacados

### threads_to_github.py
Conecta tus posts con commits. Genera actividad automática en GitHub.

```python
python threads_to_github.py
```

## Uso

```bash
# Clonar el repositorio
git clone https://github.com/codexanima1/ai-dev-tools.git
cd ai-dev-tools

# Ejecutar scripts
python scripts/ai_prompt_generator.py
python scripts/github_stats.py
```
